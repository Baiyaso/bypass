#### Domain List. 域名列表

Suffix match mode<br/>
Example:<br/>
`cn`: with match `cn`, `*.cn`, `*.*.cn`, ...<br/>
`a.com`: with match `a.com`, `*.a.com`, `*.*.a.com`, ...<br/>

後綴匹配模式<br/>
比如:<br/>
`cn`: 將會匹配 `cn`, `*.cn`, `*.*.cn`, ...<br/>
`a.com`: 將會匹配 `a.com`, `*.a.com`, `*.*.a.com`, ...<br/>
