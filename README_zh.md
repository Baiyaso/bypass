## 域名匹配规则

匹配當前域名及其所有子域名:

* `cn`: 將會匹配 `cn`, `xxx.cn`, `xxx.xxx.cn`
* `a.com`: 將會匹配 `a.com`, `xxx.a.com`, `xxx.xxx.a.com`
